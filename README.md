# DeadLine
Deadline is a learning management system designed to help students and teachers get their work done on time.

Team members: Felix Fonderie, Chinaza Maduka, MJ Dilig

if using console, open virutal environment with:
$ python3 -m venv venv
$ source venv/bin/activate
and then download dependencies in requirements.txt using pip3

if using development environment, save, close, and reopen after initially opening app in software, it should automatically read requirements.txt and download them

run.py will create an app instance and run it
runserver.py will create a local server that runs the app
